Creating a captcha

My first task for soihub.org was to create a captcha, that would hopefully stop spammers from creating accounts on the website. I created a few javascript functions, with the help of jQuery, to implement the functionality, and the captcha has been successful preventing spammers from accessing the website inappropriately.

The Original Soihub

The original site of soihub.org lacked a modern appeal. Some things on this page are tacky and dated, like the form cutting into the image in the header and the background color is slightly odd. More importantly, the website is not responsive.

Soihub Now

Soihub is in the process of moving to bootstrap. Most of its pages are using Twitter's CSS framework and look modern and elegant. This page is lively and modern.

The original Soihub

You can see that soihub.org needed an update, because the webpage does not scale well. In the mobile view, only half of the page is visible. The user would have a difficult time browsing the website.

Soihub Now

Now that soihub.org uses bootstrap the user can intuitively navigate the website. The menu collapses on the mobile view, and the rest of the page is flawlessly displayed.

Creating a captcha

When my boss first told me to create a captcha for soihub.org, I created a few javascript functions, with the help of jQuery, to implement the functionality, and the captcha has been successful preventing spammers from accessing the website inappropriately.

Soihub Now

Soihub Now

You can see that soihub.org needed an update, because the webpage does not scale well. In the mobile view, only half of the page is visible. The user would have a difficult time browsing the website.

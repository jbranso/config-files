Creating a captcha

When my boss first told me to create a captcha for soihub.org, I created a few javascript functions to implement the functionality, and the captcha has been successful preventing spammers from accessing the website inappropriately.

My first task for soihub.org was to create a captcha,

that would hopefully stop spammers from creating accounts on the website. After I uploaded this captcha to the website, spammers have been prevented from creating accounts.



This is the original version of soihub.org.

It is not using bootstrap. The design is slightly dated, and the website is not responsive.



After I created the captcha, I began to port the website to using

Twitter's bootstrap framework. This is the result of my efforts. The site's design is modern and appealing.
